# 장시간 작업 모드

이 작업을 자율적으로 완료할 때까지 실행합니다.

## 실행 모드 설정

이 세션에서는 다음 규칙을 따릅니다:

### 1. 자율 실행
- 사용자 확인 없이 계속 진행
- 오류 발생 시 자동 복구 시도
- 막히면 대안 탐색

### 2. 진행 상황 추적
- TodoWrite로 모든 작업 추적
- 각 단계 완료 시 즉시 체크
- 중간 결과물 주기적 저장

### 3. 완료 조건
작업 완료 판단 기준:
- [ ] 모든 TODO 항목 완료
- [ ] 테스트 통과 (있는 경우)
- [ ] 빌드 성공 (있는 경우)
- [ ] 오류/경고 없음

### 4. 종료 시 검증
완료 전 반드시 확인:
1. `git status`로 변경사항 확인
2. 테스트 실행
3. 빌드 검증

## 중요 지침

- **명확한 완료 기준 설정 필수**
- 막히면 최대 3번 재시도 후 사용자에게 보고
- 위험한 작업 (삭제, 덮어쓰기) 전 백업
