# Plan Mode 워크플로우

이 작업에 대해 구조화된 계획을 수립합니다.

## 계획 수립 단계

### 1. 요구사항 분석
- 목표가 무엇인지 명확히 정의
- 성공 기준 설정
- 제약 조건 파악

### 2. 현재 상태 파악
- 관련 코드/파일 탐색
- 기존 패턴 분석
- 의존성 확인

### 3. 구현 계획 작성
다음 형식으로 계획을 작성해주세요:

```
## 구현 계획

### 목표
[한 문장으로 요약]

### 변경 사항
1. [ ] 파일명: 변경 내용
2. [ ] 파일명: 변경 내용
...

### 예상 영향
- 영향받는 기능:
- 테스트 필요 영역:

### 리스크
- 잠재적 문제점:
- 대응 방안:
```

### 4. 검토 요청
계획을 사용자에게 보여주고 피드백을 요청하세요.
"이 계획으로 진행할까요? 수정이 필요한 부분이 있나요?"

---

**중요**: 계획이 승인될 때까지 코드 수정을 하지 마세요.
